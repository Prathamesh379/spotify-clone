# Spotify Clone

A responsive Spotify user interface clone built using HTML and CSS.  
This project focuses on layout design, sidebar navigation, cards, and a music player UI.

## Features
- Spotify-inspired layout
- Sidebar navigation
- Music player UI
- Responsive design
- Clean card-based sections

## Technologies Used
- HTML5
- CSS3
- Font Awesome
- Google Fonts
